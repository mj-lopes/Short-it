Encurtador de links

Ensira o link para ser encurtado

  • add_circle_outline

    Ao adicionar a URL, a mesma é mandada ao back-end...

  • save

    Que é salva na database, e gerado um link menor através de um hash.

  • search

    Ao abrir o link, é realizado uma busca na database, o qual é realizada a procura pelo o link original...

  • send

    Que ao ser encontrado, é devolvido ao navegador para que o mesmo redirecione o usuário ao local do link original.
